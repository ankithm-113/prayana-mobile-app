### "ಕರ್ನಾಟಕದ ಅನೇಕ ಲೋಕಗಳಲ್ಲಿ ಪ್ರಯಾಣ" — Travel through the many worlds of Karnataka.

**Prayana** is a premium, AI-powered travel companion built exclusively for the state of Karnataka. It isn't just a travel app; it's a bridge between the traveller and the 31 unique districts, rich heritage, and hidden culinary secrets of the state.

---

## ✨ Features that Define Prayana

### 🤖 AI-Powered Smart Planner
Generate a complete, personalized itinerary in under 30 seconds for just **₹9**. Tell the app your starting point, duration, budget, and interests (Heritage, Nature, Food, etc.), and watch your perfect trip materialize.

### 🗺️ My Karnataka (District Explorer)
A gamified "fog-of-war" map covering all 31 districts. Explore new areas to light them up on your personal dashboard, earning streaks and rewards along the way.

### 🍽️ Hidden Eats (District-by-District)
Discover no-signboard groundnut stalls, legendary Udupi breakfast joints, and authentic Coorg kitchens verified by real travellers. 

### 🚂 Kanasu Daari (Dream Routes)
15 iconic Karnataka corridors where the journey is the destination. Get contextual alerts on which side of the train to sit for the best views in Sakleshpur or the Maravanthe beach stretch.

### 🛡️ One-Tap SOS Safety
A dedicated emergency sheet connecting you instantly to the **Police (100)**, **Ambulance (108)**, and **Karnataka Tourism Helpline**, with real-time GPS coordinates.

---

## 🛠️ Technical Architecture

- **Core**: React Native (Expo)
- **State Management**: Zustand (Multi-store architecture for Streaks, Auth, Promo)
- **Backend**: Firebase (Firestore, Authentication, Functions)
- **AI Engine**: Claude AI / OpenAI integration for intelligent routing
- **Design System**: Custom high-fidelity primitives using **DM Sans** & **Playfair Display**

## 🚀 Getting Started

1. **Clone the project:**
   ```bash
   git clone https://github.com/Ankith-ms/Prayana.git
   ```
2. **Install dependencies:**
   ```bash
   cd Prayana
   npm install
   ```
3. **Run in Development:**
   ```bash
   npx expo start
   ```

## 🎨 Visual Identity

Prayana's design is inspired by the landscapes of Karnataka:
- **Prayana Red (#C8102E)**: The red soil of Hampi and the vibrancy of Yakshagana.
- **Karnataka Gold (#F5C518)**: The golden fields and brass temple bells.
- **Deep Coffee (#1A1208)**: The rich soil of Chikmagalur.
